WEB DESIGN

Šta je web design?

Web design je u suštini dizajniranje web stranica. Pomoću njega mi pravimo samu organizaciju web stranice, njenu koncepciju i njene detalje. Web design nam je veoma bitan iz razloga jer nam on govori mnogo o samom sadržaju stranice, ali isto tako i o osobi koja je radila sami taj design i kompletnu stranicu. Neadekvatan web design neke stranice može biti vrlo loš za korisnike i čitatelje jer ne zadržava dovoljno pažnje, a ujedno nemaju dovoljan sadržaj da se informišu i dobiju konkretne informacije, kao i sam uticaj na firme koje posjeduje web stranicu, jer je to vid njihove reklame, odnosno promovisanja.

Elementi web desin-a

Prave web stranice koje vidimo u 90% slučajeva na internetu su kucane u HTML-u. Moram naglasiti da HTML nije programski jezik, već je to jezik za označavanje i koji dozvoljava da u njemu kreiramo sadržaj odnosno sami izgled stranice, stavljajući u njega odlomke, naslove, slike i mnoge druge stvari koje su nam potrebne da bi naša stranica bila potpuna. Također, razni elementi su nam potrebni:

  • HTML – označavanje;
  • CSS - uređivanje;
  • JavaScript - funkcionalnost;
  • Multimedija;
  • MySQL - baza podataka;
  • Organizacija;
  • PHP

Historija web design-a (1988 - 2012)

1980.godine, čovjek zvani Tim Berners-Lee koji je stvorio globalni projekat hiperteksta, koji se kasnije zvao World Wide Wibe i služio je za informacije. Isto tako, ovo je razlog za početak protokola koji se zove HTTPD. 1993.godine kreiran je preglednik koji se zove Mosaic, a kreirali su ga dva naučnika Marc Andreessen i Eric Bina. Godinu dana kasnije, kreirana je nova verzija Mosaic-a koja se zove Mosaic Communication Corp. Trenutno taj preglednik je piznat kao Netscape Communications.

Mogu samo reći da od 1996 do 1999.godine je postojalo mnogo verzija HTML, kao i same JavaScript. Ali, 2000-te godine se desila velika promjena, jer je došlo do prevelike upotrebe web design-a, što je uticalo da se verzije HTML, CSS i JavaScript poboljšaju, što je i uspjelo. Jer, danas imamo mnogo olakšica za izradu web stranica u ovim jezicima, kao i brzinu i jednostavnost.

Napredak web design-a

Sami napredak web design-a je omogućio korisnicima da prenesu svoju maštu i kreativnost na web stranicu i tako svakog korisnika ispuni zadovoljnim, uspješnim i sretnim, jer je svoju misao prenio na kompjuter. Prije nije bilo toliko izvodljivo, jer dosta stvari u HTML i CSS su bile teške. Naravno, s vremenom, napredkom i razvojom došli smo do vremena gdje je HTML puno kompleksniji, ali mnogo jednostavan za naučiti i za upotrebu. Sada se korisniku puno više daje, puno više mu se uslova ispunjava nego prije. Neki od tih uslova je da se ubaci slika, tekst, lista, video, tabela i mnogo drugih raznih opcija, koje se mogu odmah urediti u samom kodu. Mogu još reći da je i CSS napredovao, pa ga koristimo u kombinaciji sa HTML, gdje kompletan sadržaj stranice posebno uređujemo u CSS, a ne u HTML. Iako ima ljudi koji prakticiraju odmah uređenje u HTML-u.

Razvoj web design-a

Kada je riječ o razvoju web design-a, ne smijemo zaboraviti stručnjaka, čovjeka bez kojeg razvoj web design-a ne bi bio moguć. Njegovo ime je Jacob Nielsen. Zbog njega koji je omogućio taj razvoj, dolazimo da se možemo nadovezati na prethodni dio, gdje sama izrada web stranica postaje lakša i jednostavnija za upotrebu, pored njene kompleksnosti. Sve to nekako sadrži prostu navigaciju koja je pogodna za svakog od nas, koji stvaramo, koji gledamo, koji koristimo, a i za one koji čitaju.

Četiri glavna razdoblja u web design-u

Generacija Opis
I Generacija Mosaic je prvi preglednik koji je kreiran, HTML se razvija, i postaje popularniji i napredniji.
II Generacija Napredovanjem HTML i razvojom gdje su se omogućile minimalnije stvari za izradu web stranica, je došlo do prekomjernog širenja i upotrebe web design-a, jer je olakšalo i pjednostavilo korisnicima upotrebu.
III Generacija Ali ponovnim napredovanje, gdje se korisnicima pruža mogućnost ubacivanje slike, snimki, animacija, je još više zainteresovalo korisnike, za boljom izradom, kao i samom privlačenjem klijenata na njihovu web stranicu.
IV Generacija Sada je to mnogo naprednije, jer možemo pronaći razne web stranice koje imaju svrhu za naručivanje raznih stvari putem interneta, za prodaju i slično. Tim napredovanjem je korisnicima omogućilo zadovoljstvo i vrlo jaku funkcionalnost, a klijentima uštedu vremena za trgovine, kao i pažnju i priliku za viđenjem novih stvari.
